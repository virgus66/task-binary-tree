# Skygate - JS Coding Challange
Simple front-end based Form builder

## Technologies used
* Yarn
* Webpack
* React
* Scss

## Instalation 
```sh
git clone https://github.com/WitoTV/skygate-recruitment-task.git [folder-name]
cd [folder-name]
yarn install
```

## Building
```sh
yarn run build
```

## Development
```sh
# Run development mode on localhost:9000
yarn start
```
